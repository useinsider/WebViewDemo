## 📌 Description

This project is a simple **Android & iOS demo application** showcasing how to integrate and use basic features of the **Insider WebView SDK**.  
The example includes how to setup the WebView, setup **Insider SDK** on it, and load a web page on it. It also includes example implementations of commonly used SDK methods to help you get started quickly.
